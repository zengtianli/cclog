# cclog

Browse, summarize, and review your Claude Code sessions.

## Install

```bash
pip install cclog
```

## Quick Start

```bash
# Build session index
cclog index

# List recent sessions
cclog list

# Filter by date
cclog list --date 2026-03-29

# Show session details
cclog show <session-id>

# Usage statistics
cclog stats
```

## Requirements

- Python 3.11+
- Claude Code (for session data and AI summarization)
